## 🎯 项目概览

**审计底稿引擎**是一个基于节点图的可视化审计工作流平台，支持AI智能分析、实时协作和灵活的插件扩展。

### 核心价值

```
🎯 可视化工作流编辑   - 拖拽式零代码操作
📊 智能数据分析       - 15种专业节点
🤖 AI辅助审计         - GPT/Claude集成
📁 文件管理           - 多格式支持
⚡ 实时执行监控       - 进度日志追踪
👥 团队协作           - 权限管理
```
